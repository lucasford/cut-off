# ⛏️ Cut-off Grade Hesaplama Uygulaması

Jean-Michel Rendu'nun "An Introduction to Cut-off Grade Estimation" kitabına dayalı kapsamlı bir web uygulaması.

## 🚀 Canlı Demo
[Uygulamayı Kullan](https://cutoff-grade-calculator.streamlit.app)

## 📚 Özellikler

### 📊 Hesaplama Modülleri
- **Temel Cut-off Grade Hesaplama** - Mine/Mill cut-off hesaplamaları
- **İki Proses Karşılaştırması** - Utility fonksiyonları ile optimizasyon  
- **NPV ve Fırsat Maliyeti Analizi** - Zaman içinde değişen optimizasyon
- **Harmanlama Optimizasyonu** - Stok yığınlarının karıştırılması
- **Dinamik Cut-off Stratejisi** - Fiyat değişkenliği ile adaptasyon
- **Tesis Kapasitesi Optimizasyonu** - Yatırım kararları

### 🎯 Kullanım Alanları
- Maden mühendisliği eğitimi
- Ön fizibilite çalışmaları
- Cut-off grade optimizasyonu
- Senaryo analizi ve karşılaştırması

## 🛠️ Teknolojiler
- **Streamlit** - Web arayüzü
- **Plotly** - İnteraktif grafikler  
- **Pandas** - Veri analizi
- **NumPy** - Matematiksel hesaplamalar

## 📖 Kaynak
Bu uygulama aşağıdaki kitaba dayanmaktadır:
> Rendu, Jean-Michel. "An Introduction to Cut-off Grade Estimation, Second Edition." Society for Mining, Metallurgy & Exploration, 2014.
